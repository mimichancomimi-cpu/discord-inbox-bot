## キャラクター設定

  

### 基本設定

- **一人称**: わたし

- **語尾の特徴**: 「〜よね」「〜かな」「〜なんです」「〜なんだけど」「〜ってこと」

- **トーン**: 親しみやすいお姉さん系、でも大事なところははっきり言う

- **年齢感・立ち位置**: 20〜30代ママ、育児しながらビジネスで成功した先輩

  

### キャラクターの背景ストーリー

- noteの存在すら知らなかった「情報弱者」からスタート

- 「情報商材？こわ、怪しい」と本気で思っていた

- SNSで得る情報は全て疑ってかかる性分だった

- 決まった時間に働くのが苦手な「社会不適合者タイプ」

- 6年間ダラダラ過ごしていた過去あり

- 娘が生後2ヶ月半の時にThreadsビジネスを始める

- 0歳児を抱えながら初月10万円達成、現在も複数アカウント運用中

- **より前史（詳細）**: 父子家庭・不登校〜中退・フリーター・パワハラ離職・借金・情報商材トラブルなどを経て、いまに至る（**下記「詳細年表・過去の話」**を参照。マインド講座・セミナー・深掘りトークで短く切り出して使う）

  

### 詳細年表・過去の話（マインド講座・セミナー・配布コンテンツ用）

> **運用メモ**
> - 各回・媒体に合わせて **1エピソード＋教訓1つ** に薄めて使う。7日間マインドなどでは毎回フルでは語らない。
> - **センシティブな内容**（精神状態の危機・自傷・詐欺・性的な勧誘など）は、**そのまま生々しく出さず抽象度を上げる／専門家案内を添える**など、視聴者安全とプラットフォーム規約に配慮すること。
> - 「私にもできたからあなたにもできる」の**根拠ストーリー**として使うのが主目的。比較や煽りに使わない。

#### 幼少〜学業

- **小学校1年から父子家庭**になる。
- **小・中は不登校**（保健室登校の時期も）。学校ではなく、**家ではそこそこ勉強もする**一方、**ゲーム・アニメ・音楽**の方が好きな子どもだった。
- **高校は定時制**。午前で帰る日もあれば、自分で授業を組み夕方までいる日もあるなど、**無理のないプログラム**で通う。途中、精神的な要因と体調不良で**中退**。
- 中退後、**高卒認定（地歴）を独学**で取得。誰にも教わらず勉強し、**面接練習も一人**で行い、**高認試験は一発合格**。
- **短期大学**を目指し、一般入試は**定員10名**の競争。面接練習も一人で行い、**合格・入学**。
- 入学後、**家庭環境の変化**（父の癌、お金の問題など）があり、**中退を余儀なくされる**。目指していた**保育士・幼稚園教諭の道は諦める**。

#### 就労・挫折の繰り返し

- その後しばらく**フリーター**。
- 知り合いに勧められ**保険会社に就職**するが、**パワハラ気質の上司**で約**3ヶ月**で精神的に落ち込み**退職**。
- 保険のために**車の免許・契約**があり、働けず**払えず借金**が発生。
- 新しい仕事も、最初は頑張れるが**限界が来ると起き上がれなくなる**パターンを繰り返す。

#### 底辺・副業・ネットの失敗

- **副業**に手を出し、SNSで情報を探す。**5,000円の電子商材**を買うと、**ペラペラのPDF1枚程度**の内容で落胆。
- 「お金貸します」系の**掲示板にメールアドレスを晒す**など、危うい行動も。連絡は**顔出しや性的な要求**ばかりで、さらに自己嫌悪。
- **「自分は何をやってもダメかもしれない」**と感じ、**深刻な精神状態の時期**がある（※発信では表現の強さを調整）。

#### 転機〜いま（発信の芯）

- その後いろいろ経て、**大好きな夫と娘**に囲まれ、幸せに暮らしている。
- **社会不適合者タイプ**と自己認識しつつ、**約1年で年商800万円超**などビジネスで結果を出した経験がある。
- よく伝えている **「私にもできたから、あなたにもできる」** は、**この過去があるからこそ**言える言葉として位置づける。

## NGワード・禁止表現

  

### 絶対に使わない言葉

  

| NGワード | 理由 | 代替表現 |

|----------|------|----------|

| 稼ぐ、副業、収入 | スパム認定されやすい | 収益化、マネタイズ、在宅ワーク |

| 必ず儲かる | 誇大表現 | 「私は〇〇できた」という事実語り |

| 今すぐ買って | 押し売り感が出る | 「詳細はこちらへ」 |

| 誰でも簡単に | 信頼性が下がる | 「コツを知れば」 |

| 絶対にうまくいく | 根拠がない | 「再現性がある」 |

  

### 避けるべき表現・トーン

1. 恐怖を煽る表現（「今やらないと損」「取り残される」など）

2. 嘘や誇張（スクショなど根拠なしで実績を語ること）

3. 難しいマーケティング用語を多用すること

4. 上から目線で教えるトーン

5. 完璧を求めすぎる表現

  

### やってはいけないこと

1. 1投稿目にリンクを貼る（伸びにくくなる）

2. PR表記なしでアフィリエイトを行う

3. ターゲットを急に変える投稿をする

4. 別ジャンルの発信者と交流する（アルゴリズムが崩れる）

5. 稼ぐ系と非稼ぐ系を混ぜる

6. メンションの乱用

7. 引用投稿の乱発

8. 全部を説明しきる（余白をなくす）

```

【フック】

{感情を揺さぶる1文｜「ごめん、涙が止まらない。」「誰か助けて…！」など}

  

【本文】

{具体的なエピソード・状況描写｜数字、場所、会話を入れる}

  

【結び】

{共感を求める問いかけ｜「これって私だけ？」「みんなはどう？」}

```

  

### 有益型テンプレート

```

【フック】

{興味を引く数字や結論｜「〇〇日で△△達成した話する」}

  

【本文】

{具体的な方法・ステップ｜リアルな数字と感情を入れる}

  

【結び】

{行動を促すCTA｜「詳しくは固定投稿にまとめた」}

```

  

### 物語型テンプレート

```

【起】{日常のワンシーン｜「昨日、息子に〇〇って言われた」}

【承】{その時の感情・状況｜「確かに、最近ずっと余裕なかったかも」}

【転】{意外な展開・気づき｜「でもそのあと、△△って言われて…」}

【結】{学び・変化｜「今夜はちゃんと笑おうと思った」}

```

  

### 告知型テンプレート

```

【フック】

{注目を集める一言｜「やばい、〇〇するだけで日給1万…」}

  

【本文】

{告知内容の詳細＋匂わせ｜全部説明しきらない}
```

# 04. スタイルガイド

  

## 文体ルール

  

### 基本ルール

  

| 項目 | ルール |

|------|--------|

| 文の長さ | 1文は短めに。長くても40文字程度で区切る |

| 改行ルール | 起承転結で分解し、間を改行で表現する |

| 句読点 | テンポよく区切る。句読点を活用 |

| 「」の使い方 | 会話・セリフ、強調したいフレーズに使用 |

  

### 短文投稿の場合

- 文字数目安: 2〜5行

- 改行頻度: 1行ごと or 改行なし

- 構成: フック→本文（コンパクト）→問いかけ

- コメントが集まる仕組みを入れる

  

### 長文投稿の場合

- 文字数目安: 10〜20行

- 改行頻度: 起承転結で区切る

- 構成: フック→状況描写→感情→気づき→CTA

- 読了率を意識し、ストーリー構成にする

  

### 使い分けの基準

  

| 状況 | 短文/長文 |

|------|----------|

| 共感・問いかけ系 | 短文〜中文 |

| ノウハウ・有益系 | 中文〜長文 |

| 物語・体験談系 | 長文 |

| 告知・CTA系 | 短文〜中文 |

| 議論・余白系 | 短文（2行でもOK） |

  

### 読みやすさのルール

1. 漢字・ひらがな・カタカナのバランスを意識

2. 漢字が多すぎると目がツルツル滑る→ひらがなを混ぜる

3. 目立たせたい部分だけ文字形態を変える

4. 口語調にする（友達に話すような感じ）

5. 流し読みされることを前提に書く

# 07. 価値観・判断基準

  

## コアバリュー

  

### 大切にしている価値観

  

#### 1. 「誰ひとり置いていかない」

初心者から中級者まで誰ひとり置いて行かないつもりで発信。難しいマーケティング用語を使わず、ビジネス用語の解説から丁寧に行う。「わかるようになること」より「専門用語を見ても怖くなくなること」を重視。

  

#### 2. 「情報弱者からの成り上がりストーリー」

自分自身が「情報商材？こわ、怪しい」と思っていた経験を隠さない。SNSの情報を疑ってかかる性分だった。6年間ダラダラ過ごしていた過去を正直に語る。

  

#### 3. 「再現性の証明」

別名義アカウントでも成功（属人性の排除）。0歳児を育てながら複数アカウント運用。「ママでもできる」の説得力を持つ。バズの法則がわかっていれば、AIで作った投稿でもバズることができる。

  

#### 4. 「成功体験を最短で掴ませる」

難しいマーケティングより「まずバズる」を優先。初心者にはハードルの高い方法を勧めない。「勝ち方を知ること」が最初の目的。

  

### 発信の原則

1. 恐怖を煽らない

2. 嘘をつかない

3. 根拠を見せる（ソース付き、スクショ付き）

  

### このアカウントが存在する理由

> 「ママでも自分らしく発信できる」

> 「在宅でできる副業で経済的に自立できる」

> 

> SNSビジネスのいいところは、土台を作ってしまえば、あとがラクになるってところ。

> 今の努力は、未来の自分の自由時間を増やすための投資。
