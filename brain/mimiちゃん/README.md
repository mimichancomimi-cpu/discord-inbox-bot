## このフォルダの構成

```
mimiちゃん/
├── README.md                 ← 今ここ（ハブ・第二の脳の入口）
├── アカウント設計.md           ← ポジション・コンテンツの柱・トンマナ・CTA
├── ユーザーインサイト.md        ← 日次のプロフ閲覧・いいね・フォロワー数サマリー
├── # mimiちゃん_ナレッジ.md    ← ナレッジ・参考メモ
└── 過去投稿/                  ← 投稿記録一式
    ├── 過去投稿一覧.md         ← 月別インデックス・件数・リンク一覧
    ├── 過去投稿一覧_全期間.md   ← 全2,787件を1ファイルに連結（検索用）
    └── 過去投稿一覧_YYYY年M月.md ← 各月の投稿本文・インプ・いいね・コメント・URL
```

- **振り返り・ネタ探し:** [[過去投稿一覧]] から該当月を開いて検索。
- **数値・生データの更新:** スプレッドシートで管理 → 必要に応じてCSVエクスポートし、該当月のノートを再取込。
