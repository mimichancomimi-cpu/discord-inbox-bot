# mimiメソッド 章ごとの内容構成ルール

新規追記・新章作成時に、既存の章と同じ構成で書くための分析まとめです。

---

## 1. 見出しの表記ルール（全章共通）

| レベル | 形式 | 例 |
|--------|------|-----|
| **章** | `## 【第Ｎ章：〇〇編】` | `## 【第１章：はじめに・導入編】` |
| **節** | `## ▼N-M タイトル` | `## ▼4-1 アカウント設計の全体像と目的` |

- 節のハイフンは**半角** `-`
- 節の番号とタイトルの間は**スペース1つ**（コロンは使わない）

---

## 2. 章タイトル直下の「章リード」のパターン

章（`## 【第Ｎ章：〇〇編】`）の直後に何を置くかは、現状で章によって異なります。追記時にどれに合わせるか選べるように一覧にしました。

| 章 | 章リードの有無・内容 | 使い分けの目安 |
|----|----------------------|----------------|
| **第1章** | なし（空行のあとすぐ ▼1-1） | 導入・短い章向け |
| **第2章** | 空行 → **テーマ：**（1行）→ 補足1行 → **※重要性を伝える** → 空行 | 「なぜこの章が大事か」を伝えたいとき |
| **第3章** | なし（空行のあとすぐ ▼3-1） | シンプルに節から始めるとき |
| **第4章** | 空行 → **目的：**（改行）→ 複数行で章の目的を説明 → 空行 | 章の「目的」を箇条書き風に示したいとき |
| **第5章** | **1行の説明文**（章の内容を要約）→ 空行 | 章のコンセプトを1文で示したいとき |
| **第6章** | なし（空行のあとすぐ ▼6-1） | 章リードなしでOKなとき |
| **第7章** | なし（空行のあとすぐ ▼7-1） | 同上 |

**追記時の推奨**  
- 新章で「この章で何を学ぶか・なぜ重要か」を明示したい → **第2章型（テーマ：＋※重要性を伝える）** または **第4章型（目的：複数行）**  
- 1文で雰囲気を伝えれば十分 → **第5章型（1行説明）**  
- 章リードは不要 → **第1・3・6・7章型（なし）**

---

## 3. 節（▼N-M）直下の「節リード」のパターン

節見出しの直後に、「テーマ：」を置くかどうかです。

| 章 | 節の傾向 | 例 |
|----|----------|-----|
| **第1章** | テーマなし。空行のあと「1.小見出し」から開始 | ▼1-1 → 1.講座の特徴 |
| **第2章** | **テーマ：** あり（1行）→ 空行 → 1.小見出し | テーマ：まずは「言葉の壁」をなくして… |
| **第3章** | テーマなし。空行 → 1.小見出し | ▼3-1 → 1.リサーチの基礎理解 |
| **第4章** | 多くはテーマなし。空行 → 1.小見出し | ▼4-1 → 1.アカウント設計編 |
| **第5章** | 節によって**テーマ：**あり／なし | ▼5-1 はテーマあり、▼5-2 はなしなど |
| **第6章** | テーマなし。空行 → 1.小見出し | ▼6-1 → 1.運で伸ばさず、検証で伸ばす。 |
| **第7章** | **テーマ：** あり（1行）→ 空行 → 1.はじめに など | テーマ：Threads発信を「収益の流れ」に… |

**追記時の推奨**  
- 節の「ねらい」を一言で示したい → **テーマ：** を入れる（第2章・第7章型）  
- 本文からすぐに入りたい → テーマなしで「1.小見出し」から（第1・3・4・6章型）

---

## 4. 節の中身の構成（小見出し・本文）

- **小見出し**：`1.〇〇` `2.〇〇` `3.〇〇` … のように**数字＋ピリオド＋スペース＋見出し**
- **最後**：多くの節で「まとめ」がある（`6.まとめ` や `〇.まとめ`）
- **箇条書き**：`・` または `①` `②` を使用
- **空行**：見出しのあと・段落のあとは空行または「  」（スペース2つ）で区切る

---

## 5. 追記案を書くときのチェックリスト

- [ ] 章見出しは `## 【第Ｎ章：〇〇編】` になっているか
- [ ] 節見出しは `## ▼N-M タイトル`（半角ハイフン・番号とタイトルの間にスペース）か
- [ ] 章リードを入れる場合、第2章型／第4章型／第5章型のどれに合わせるか決めたか
- [ ] 各節の直下に「テーマ：」を入れるかどうか、章内で揃えたか
- [ ] 節内の小見出しは「1.〇〇」「2.〇〇」…の形式か
- [ ] 必要に応じて最後に「〇.まとめ」を入れているか

---

## 6. 章別・節数・章リード一覧（参照用）

| 章 | 節数 | 章リード | 節のテーマ |
|----|------|----------|------------|
| 第1章 はじめに・導入編 | 2（▼1-1, ▼1-2） | なし | なし |
| 第2章 基礎編 | 6（▼2-1〜▼2-6） | テーマ：＋※重要性を伝える | あり |
| 第3章 リサーチ編 | 6（▼3-1〜▼3-6） | なし | なし |
| 第4章 アカウント設計編 | 6（▼4-1〜▼4-6） | 目的：（複数行） | なし |
| 第5章 投稿作成編 | 7（▼5-1〜▼5-7） | 1行説明文 | あり／なし混在 |
| 第6章 運用編 | 5（▼6-1〜▼6-5） | なし | なし |
| 第7章 マネタイズ編 | 6（▼7-1〜▼7-6） | なし | あり |

新規で「第8章」などを追加する場合は、上記のいずれかのパターンに揃えると、既存のmimiメソッドの読み心地と統一できます。
