# 個別サポートLP — UTAGE組み立て手順

`brain/mimiメソッドLPソースコード.md`（本番の mimiメソッドLP HTML）の**色・帯のリズム**（グラデ、淡いピンク `#fff0f5`、赤系CTA）を参考に、**個別サポート用**のLPを**カスタムHTMLを複数ブロックに分割**したものです。

## 事前準備（1ページにつき1回）

1. **`../ランディング_サポート売る_UTAGE用_カスタムCSS.css`** の中身を**すべて**コピーし、該当UTAGEページの **「カスタムCSS」** に貼り付けて保存する。  
   - グラデヒーロー（`.hero--mimiLP`）、淡いピンク帯（`.band-lp-soft`）、チェックボックス風（`.lp-check-box`）、赤帯（`.cta-strip`）などはこのCSSに含まれます。

## カスタムHTMLの並べ順（上から）

| 順番 | ファイル | 内容 |
|------|------------|------|
| 1 | `01_ヒーロー.html` | グラデ・キャッチ・リード文 |
| 2 | `02_こんな方へ＋導線.html` | ターゲット＋チェック風リスト＋「声へ」ボタン |
| 3 | `03_サポートした方の声.html` | 33枚サムネギリッド（要約キャプション付き） |
| 4 | `04_個別サポートでできること.html` | 4カード＋注意書き |
| 5 | `05_変化のイメージ.html` | 実績例リスト |
| 6 | `06_FAQ.html` | よくある質問 |
| 7 | `07_CTA＋フッター.html` | 赤帯ラベル・CTA・編集メモ・フッター |

各ファイルの**コメント行は消してOK**（UTAGEにそのまま出したくない場合）。

## 本番LPとの役割分担（参考）

- **mimiメソッド本番LP** … VSL・教材の価格・決済・つきのわセット等は **UTAGEの標準パーツ**（動画・決済フォーム・ボタン）で組む想定。
- **この個別サポートLP** … **カスタムHTML＋上記CSS** で訴求・声・FAQまで。  
  **お申し込み・決済**は `07` のボタンURLを差し替えるか、**別セクション**に UTAGE の「決済」「フォーム」を置いても問題ありません。

## 一括コピペ用（従来）

すべて1つのブロックにしたい場合は、引き続き  
`../ランディング_サポート売る_UTAGE貼り付け用.html` の `<div class="mimi-utage-lp">` ～ 閉じタグまでを利用してください。

## アンカーリンク

- `02` のボタンは `#voices`（パーツ03のセクションID）へスクロールします。
- `07` のCTAブロックのIDは `id="cta"` です。`02` のテキストから参照しています。
