## トピック1: AI生成投稿は伸びないのか？（雑談・冒頭）

- 「AIで作った投稿が伸びない」「アルゴリズムに弾かれてる」という噂について議論
- mimiちゃんの見解：
  - 3〜5行の短い投稿ならAIそのままでも全然平気
  - 長い投稿だとAIっぽさが出やすい（ビジュアルカードは特に分かりやすい）
  - 単純に投稿の質が悪いだけの可能性が高い
  - AIに頼りすぎず、自分で作った方がいい
- **ジャンル認知の重要性：** 投稿ごとにジャンルが認識されるので、ジャンルに合った単語を入れることが大事
  - 例：復縁系 → 毎回「復縁」と入れる / ママ系 → 「ママ」「子供」「娘」「息子」等
- AIで作る場合もリライト時に「何の投稿か分かる単語が入っているか」をチェックする
- AIっぽくない投稿を見る目を養ってほしい

## トピック2: 稼ぐ系 vs 非稼ぐ系（ひかせぐ系）

- **稼ぐ系：** お金を稼ぐ方法を教えるジャンル
  - SNSでの稼ぎ方、ポイ活、クライアントワークなど
  - ビジネスアカウントと呼ばれることもある
- **非稼ぐ系（ひかせぐ系）：** 稼ぐ手法を発信しないジャンル
  - 占い、恋愛、自己啓発、スピリチュアル、楽天アフィなどすべて
  - 稼ぐ以外の価値を提供している（おすすめ商品・悩み相談・マインド系など）
- **楽天アフィの注意点：** 楽天アフィは「非稼ぐ系」なので、商品紹介アカウントで「アフィリエイトで稼いでます」と言うのはNG
  - フォロワーが「お金のために商品紹介してたんだ」とショックを受ける
  - 住み分けが大事：ビジネスアカウント（同業者向け）と商品紹介アカウントは分ける

## トピック3: ローンチ（プロダクトローンチ）

- 正確には「プロダクトローンチ」＝マーケティング用語
- 商品を販売すること。リリースに近い意味
- ただし単に販売開始だけでなく、**興味付けの段階から目標達成まで**の全体の流れを指す
  - 興味付け開始 → 販売 → 目標部数・金額達成まで
- 簡単に言えば：「ローンチ」と聞いたら「商品販売するんだな」と思えばOK

## トピック4: アルゴリズム

- SNSやプラットフォームの投稿表示の仕組み・ルール
- 「誰に、どれぐらいの頻度で表示するか」の仕組み
- 明確に公開されているわけではなく、みんなが傾向を研究して仮説を立てている
- 例：リンクを1投稿目に貼ると外部露出が減る → Threadsがアプリ滞在時間を伸ばしたいから
- 「アルゴリズムを攻略する」＝ どうやったら一番伸びるかを分析・実践すること
- 自分で使うことは少ないが、発信者がよく使う言葉なのでニュアンスで覚えておけばOK

## トピック5: シャドーバン

- **垢BAN（レッドカード）：** アカウント凍結・ロックで使えない状態
- **シャドーバン（イエローカード）：** BANの手前の状態
  - インプが明らかに伸びない
  - フォロワー500人いるのに表示回数が2桁
  - コメントが見えない・通知が来ない
  - フォロワーからしかいいねが来ない
  - 検索にも出なくなる
- **原因：** Threads側に「このアカウント怪しい」と思われている状態（スパム判定等）
- **対処法：** 3日間何もせず放置すると復活することが多い
- **確認方法：** トレンドワードを入れて検索に出るか確認する方法あり
- 投稿が消された・一時ロックされた後もしばらくシャドーバン状態になることがある

## トピック6: 大手

- 明確な定義はない
- ビジネス界隈では稼いでいる額が基準になりがち
- フォロワー数を基準にする人もいるが、フォロワーが多い＝稼いでいるとは限らない
- ジャンル（村）によっても基準が異なる：恋愛村、楽天アフィ村、ノート村、スレッズ運用村など
- 市場が成熟すると基準も上がっていく
- 「自分が大手だと思ったら大手」ぐらいの認識でOK

## トピック7: マインドブロック

- **マインド：** 心の状態・精神・考え方の軸。ビジネスでは仕事に対する心構え・姿勢・基本的な考え方
- **マインドセット：** 成果を出すための心構え。考え方の骨組み
- **マインドブロック：** 自分で制限をかけている状態（心の壁）
  - 「自分には無理」「発信は向いてない」→ 可能性を閉ざしてしまう
  - 本来ポテンシャルもスキルもあるのに思い込みで成果が出ない状態
- 具体例：まだ始めてもいない段階で「税金どうしよう」「扶養から外れるのでは」と不安を探してしまう
- マインドブロックを壊す（外す）ことがビジネスで成果を出すために重要
- 発信者としてはそれを壊してあげるのも仕事
- 本を読んでみるのも手っ取り早い解決策

## トピック8: リストを取る（リストマーケティング）

- **リスト：** 顧客リスト（見込み顧客 or 購入者のリスト）
- リストを取っておくと、次の商品を出した時に営業できる
- リストに対して売ると売りやすい（認知＋信頼が既にある状態）
- **日常のリスト取得の例：**
  - ショッピングモールのアンケート → 電話番号・名前を取得 → 後日営業
  - 住宅展示場のクオカード配布 → 顧客情報と交換
  - LINE追加でクーポン → LINE登録 → 後日配信
  - 資料請求でプレゼント
- **リストマーケティング vs マスマーケティング：**
  - マスマーケティング：大衆・大勢に向けて認知を広げる（バズらせて売る）
  - リストマーケティング：LINEなどクローズドな場で1対1営業（対面セールスが得意な人向き）
- 占いアカウントはリストマーケティングがマスト

## トピック9: 仮想敵

- 自分の発信を良く見せるために、対比となる存在を少し下げること
- 例：「AIを使わず占いする」派 → AI占い派を仮想敵にする
- 例：ノートやってる人 → 「楽天アフィは単価が低いから稼げない」→ 楽天アフィを仮想敵にしてノートに誘導
- 個人批判とは違う。あくまでも手法
- 自分の発信を信頼させるために必要な手法
- 意見がぶつかって不安になっても、自分を信じて続ければOK

## トピック10: ポジショントーク

- その時一番自分にメリットがあることを言うこと
- 前と言っていることが変わる人もいる
- 市場の変化に合わせれば全員ポジショントークになりがち
- 金融系で元々使われる用語：自分の保有ポジションに有利なように市場に影響を与えようとする発言
- 「これ仮想敵かな？ポジショントークかな？」とタイムラインを見ると気づけるようになる

## トピック11: その他のビジネス用語（まとめ）

| 用語 | 意味 |
|------|------|
| **マネタイズ** | 収益化する方法。ノート・アフィリエイト・鑑定・クライアントワーク等 |
| **スピリチュアル（スピ系）** | 見えないものを信じる系。引き寄せ・潜在意識が代表的 |
| **セールスレター** | 商品を購入してもらうための説得文章。ノートの無料部分がこれにあたる |
| **LP（ランディングページ）** | 資料請求やプレゼント応募のためのページ。セールスレターが載っているページ |
| **インプ（インプレッション）** | 閲覧数・表示回数 |
| **インサイト** | アカウントの分析ページ（フォロワー属性・閲覧数など）。フォロワー100人以上で閲覧可 |
| **コンサル** | 1対1の指導。目標に向かってロードマップを引いてサポート。高単価 |
| **コンテンツ** | その人の思考をまとめたもの。誰にでも当てはまるわけではない。比較的安価 |
| **プロンプト** | ChatGPTなどAIへの指示文章 |
| **ロードマップ** | ステップ形式で目標達成までの順番を示したもの |
| **プラットフォーム** | サービス名。ノート・Brain・Tips・楽天・Amazon・SNS等 |
| **CTA（コールトゥアクション）** | 行動喚起。「いいねしてね」「フォローしてね」「LINE追加してね」等 |
| **MLM（マルチレベルマーケティング）** | マルチ商法・ネットワークビジネス。商品自体はいいものが多いが勧誘手法が敬遠されがち |
| **エアプ（エアープレイ）** | やってないことをあたかもやっているかのように見せること。ビジネスアカウントでは特にNG |
| **ペルソナ** | 理想の顧客像。「こういう人に買ってほしい」というターゲット |
| **ツリー投稿** | コメント欄に続きをつなげる投稿。リンクはツリーに貼るのが推奨 |
| **トンマナ（トーン＆マナー）** | 一貫した雰囲気・印象を与えるためのルール。フォント・色使い・文末表現・言葉遣い等を揃える |
| **タイムライン / フィード** | 投稿が表示される場所。TwitterではTL、Threads/Instaではフィード |
| **情報商材** | 情報自体が商品となるもの。ノート・Brain等。イメージはあまり良くないが本質的には本やスクールも同じ |
| **コンテンツアフィ** | 情報商材をアフィリエイトすること |
| **ASP（アフィリエイトサービスプロバイダー）** | 企業とアフィリエイターを繋ぐ仲介サービス（A8.net、もしもアフィリエイト等） |
| **フック** | 興味を引く冒頭部分。投稿の最初・ノートの最初3行・動画の最初3秒 |
| **エンゲージメント** | 反応率。フォロワー100人に対していいね100 → エンゲージメント100% |
| **CVR（コンバージョン率）** | 見た人の中で何人が行動（購入等）してくれたかの割合 |
| **垢BANリスク単語** | 副業・お金・詐欺に繋がりそうな言葉は伏せ字推奨。楽天アフィ・占い・BAN等も伏せるのが界隈の暗黙ルール |
| **スタエフ** | stand.fm（音声配信サービス） |
| **オプチャ** | LINEのオープンチャット |
| **FA（ファンアート）** | ファンアートの略 |

## トピック12: ノートの書き方（質問への回答）

- ぶっちゃけみんな本から知識を持ってきている
- ノートを書く前に専門の本を3〜5冊読むのが推奨されている
- それをキャラクターに合わせて噛み砕いて説明する
- 新分野に挑戦するときはリサーチが必須：本を読む・競合のノートを買う・YouTubeを見る
- 今はAIで効率化できるので稼ぎやすくなっている

## アイデア・気づき

- AI投稿が伸びない原因は「AIだから」ではなく投稿の質の問題が大きい
- ジャンル認知のためにキーワード（単語選び）が重要
- 楽天アフィは非稼ぐ系。ビジネスアカウントと商品紹介アカウントは住み分けが必要
- シャドーバン対策：3日間放置・トレンドワードで確認
- マインドブロックはビジネスの成長を妨げる最大の障壁。本を読む等で対処
- 伏せ字文化はリスクヘッジだが、隠すことで伝わらないこともある（戦略的に使い分ける）
- リサーチの効率化：ディープリサーチ機能の活用
