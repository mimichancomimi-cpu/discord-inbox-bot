## 主要テーマ：AIの使い方と向き合い方

### 1. AIは便利だが完璧ではない — 信用度は50〜60%

- AIは当たり前の存在になった（旦那やその上司、学生まで使っている。Google検索、Yahoo!知恵袋、Canva、写真加工アプリなど、あらゆるところにAIが搭載）
- だからこそ**「AIにできないこと＝自分らしさ」がますます大事な価値になる**
- AIの信頼度は50〜60%程度。80%でも多すぎるとみみは感じている
- **基本的にAIは間違えるもの**という前提で使うべき

### 2. AIが苦手なこと・間違えやすいこと

| 苦手な領域 | 具体例 |
|---|---|
| **計算・数字** | 占いの生年月日計算（数秘術・個性心理学・西洋占星術）で半分くらい間違える。確定申告の税率計算も不正確。計算はスプレッドシートに任せるべき |
| **薬機法・著作権** | 「これで○○が治ります」のような薬機法違反表現をAIは理解していない。アフィリエイト紹介時に注意 |
| **情報の鮮度** | ChatGPTは情報が古い。曜日を間違える（2024年の曜日を返してくるなど）。告知系投稿の日付に注意 |
| **ハルシネーション（造語・捏造）** | 存在しない心理学用語をそれっぽく作って返す。心理効果の名前と実際の効果が混ざる |
| **専門用語** | 専門的な話は自分で調べた方が早いし正確 |
| **ツールの操作手順** | 動画編集アプリの使い方を聞くと、存在しないメニューやボタンを指示してくる |

- **対策:** 最新情報にはPerplexityが比較的強い。ChatGPTのディープリサーチは参考文献リンク付きで信憑性あり
- **根本的な注意:** インターネット上にAIが書いた情報が増えている → それをさらにAIが参照して回答する循環が起きている

---

### 3. AI利用時の思考停止に注意

- AIとやり取りするだけで「考えた気」「やった気」になってしまう危険がある
- 初心者だけでなく、長く使っている人でもAIの成功体験が増えるほど思考停止しやすい
- **例（架空）:** 「アカウントのコンセプトを考えてきてください」と言われて、全部AIで作ったものを提出 → 「それはあなたがやることじゃないですよね」
- **例:** コメント欄に「AIに聞いてみました」と回答をコピペする人 → 内容が合っているかも判断せず貼るのは無責任。本人に聞きたいから聞いているのに意味がない
- **大事なこと:** 目的と目標を忘れずに使う。AIの出力の良し悪しの判断は難しいが、基本は疑ってかかる。自分の経験と結びつかないことはとりあえず疑う

### 4. AIは「あなたの価値を高める道具」

みみが意識しているAI活用の3パターン：

| 活用パターン | 内容 |
|---|---|
| **① 効率化** | 時短・作業スピードアップ |
| **② アイデアを広げる** | 発想の幅を広げてもらう |
| **③ 思考の整理** | 考えていることをまとめてもらう・分かりやすい文章に整えてもらう |

- 常にAIより上の立場（操作する側）でいること。対等でもAIより下でもない
- AIを「第二の脳みそ」「外付けメモリ」と捉えている。本体（自分）がなければ機能しない

### 5. みみ流・AI投稿作成の実践方法

**ノート・投稿作成時:**
- 書きたい内容は**全部自分で考える**。AIにはまとめ・整理・整えを任せる
- **最初に書きたいこと全部を一気に伝える**（タイトル・テーマ・言いたいこと・最終的な着地点を一息で）
  - こうすると自分の感情・信念がAIに最初から伝わる
  - 曖昧なまま仮に作らせると「それっぽいけど自分の考えとは違う」ものになりがち
- AIが勝手に補足・具体例・実績を入れてきたら、間違っている部分は全部指摘して修正指示を出す
- この繰り返しで**AIが自分に馴染んでいき**、修正なしで使える投稿が出るようになる

**アカウント初期の育て方:**
1. まず自分で投稿を作る
2. 反応が良かった投稿だけをピックアップしてAIに学習させる
3. AIに投稿を作ってもらう → リライトする
4. リライト後の内容もAIに学習させる（「こう直しました」と教える）
5. 繰り返すとAIがそのまま使えるレベルに育つ

**運用のコツ:**
- **1アカウント = 1チャット**で運用（投稿・ノート・LINE配信全部同じチャット内）→ 内容の相違が出なくなる
- チャットを移動する際は情報を持っていく
- **音声入力が非常におすすめ**（デプレさんの影響で導入。AI活用の性能が上がった）
  - 話し言葉が苦手な人も音声入力で自然に話し言葉になる
  - 言語化が上手でなくても「伝えようとする」ことが大事

- AIが当たり前の時代だからこそ「自分らしさ」が最大の価値になる
