## Q1. 投稿に画像はつけた方がいい？

**回答まとめ:**
- **画像推奨ジャンル:** 楽天アフィリエイト、ファッション、インテリア、占い → 目に留まりやすく伸びやすい
- **それ以外（恋愛・ビジネスなど）:** 無理してつけなくてOK
- **補足:** みみナビ（AI）に主導権を握らせないこと。AIは使う人に都合よく学習してしまう。自分の感覚で「本当にそれでいい？」と思ったら自分を優先する

### 💡 アイデア・気づき
- AIに主導権を握らせない使い方が重要（初心者ほど注意）
- 疑問に思ったことはコミュニティ（つきのわハウス）で質問すると、いろんなジャンルの知見が集まる

## Q2. Obsidian × Cursor の活用法

**回答まとめ:**
- **Cursor:** プログラミング用ソフトだが、複数AI（Gemini/ChatGPT/Claude）が搭載されている
- **Obsidian:** データの保管庫（本棚・脳みそ的な役割）
- **連携のメリット:**
  - Obsidianにデータを全部保存 → Cursorで読み込み＆指示できる
  - アカウントごとの設計やコンセプトをまとめておけば「アカウント①の投稿を10個作って」で自動生成
  - 壁打ち内容もObsidianに記録させられる
  - リアルタイムでノートを書き換えてくれる（note作成にも便利）
  - スプレッドシートへの転記 → 予約投稿まで自動化も可能
  - Cursor課金だけで複数AI使い分け可能
- **デメリット:** PCでしか使えない

### 💡 アイデア・気づき
- 複数アカウント運用者にはかなり有効
- 今までのAIの弱点（情報混在・古い記憶忘れ）を解消できる
- インプットにもアウトプットにも使える
- みみ自身もまだ勉強中。今後共有予定

## Q3. 恋愛アカウントのプロフィール文は弱い？（ゆりさん）

**相談内容:**
- テーマ「不倫で心がぐちゃぐちゃになった人の休憩所」
- note販売を見据えた時にプロフが弱いのでは？
- 過去に高額講座（再構築＋在宅ワーク）に計100万使った経験あり
- 現在は旦那と仲が復活してライブに行けるまで回復

**回答まとめ:**
- 今は再構築寄りの方向性がよさそう
- **理想の未来（旦那と仲が復活していること）を見せるべき**
- 「この人をフォローすれば理想に近づけるかも」と思ってもらう → その気持ちが高まったところでnote販売
- 今のプロフは誰にでも言えるような内容になっている → **ゆりさんだから言えること**を足す
- プロフィールは何回変えてもOK。「販売直前に変えよう」と構えなくていい
- ファンを作っておくとnoteは売れる

### 💡 アイデア・気づき
- マネタイズ目的なら「どうなりたいか（理想の未来）」をプロフに明示
- 高額講座を受けようか迷っている人への訴求は有効
- 発信しながらプロフを改善していくスタイルでOK

## Q4. 楽天アフィリエイトのアカウント2つともBAN

**相談内容:**
- 2アカウント運用、両方BAN。再審査2回通過後の3回目でBAN（異議申立て不可）
- 1日5投稿（うち1つはアフィなし共感系）、フォロワー二桁、開設1ヶ月
- 同じスマホで複数アカウント・メアドは分けている

**回答まとめ:**
- 複数アカウント作成自体は問題なし（アプリで10個まで作れる）
- **原因の推測:**
  - リンクが引っかかりやすい（楽天アフィ特有）
  - 同じIPで別アカが怪しい挙動 → 他アカにも影響
  - 再審査で送ったメアドがアカウントと不一致 → BAN
  - 顔セルフィーの異議申立てが通らないケースも
- **一番安全な対策:** 端末＋SIMを分ける
  - 端末1円、楽天SIM月1,000円程度で用意可能
- アカウント購入は使えないケースも多い
- 電話番号認証はしなくてもOK

### 💡 アイデア・気づき
- 1つのスマホ＋1つのWiFiで大量アカ運用 → 伸びなくなる・BANリスク
- みみ自身も過去に20個作って同じ経験あり → 端末分けで解決

## Q5. 育児アカウントのマネタイズ（note以外）

**相談内容:**
- 育児アカがバズってフォロワー増加中
- note以外のマネタイズ方法は？
- オプチャや公式LINEも検討中。フォロワー何人で案内開始すべき？

**回答まとめ:**
- **オプチャ・公式LINEの目的:** 商品を売れやすくするため（濃いファンを集めてセールス）
- **公式LINEの方が安全**（オプチャは荒れるリスクあり、特に育児ジャンル）
  - ただし無料は200通まで → 課金必要
- **フォロワー数で判断するのではなく、note販売タイミングに合わせて計画する**
- 育児ジャンルのマネタイズ案:
  - note（一番ハードル低い）
  - サポート・相談（1ヶ月いくらで相談し放題、電話相談1時間いくら）
  - グループ講座（5〜10対1）
  - 楽天アフィリエイト
  - ココナラで相談系出品
- **みみの戦略案:** 公式LINEで相談を受ける → 顧客データ収集 → そのニーズに向けてnote作成 → noteの特典で相談 → アップセルで講座・サポート
- ビジネス系の「オプチャ・公式LINEやるべき」投稿に流されないように

### 💡 アイデア・気づき
- note入り口 → 後ろ（講座・サポート）につなげるのが分かりやすい導線
- オプチャは限定募集（いいねくれた人だけ、ゴースト投稿で案内など）で質を保つ
- インスタの育児アカウント×オプチャの事例をリサーチすると参考になる

## Q6. バズった後の次の投稿は？

**回答まとめ:**
- **みみの場合:** フォロワー獲得を優先することが多い
  - 共感系でバズ → アカウントパワーが上がった状態でフォロワー獲得狙いの投稿 → 50%くらいの確率で増える
- **フォロワーが少ない（500〜1,000未満）:** フォロワー獲得優先
- **フォロワーがいる（500〜1,000以上）:** 商品誘導してもOK
- フォロワーが多い方が投稿はバズりやすくなる → 長期的にはフォロワー増が有利
- **両方やるのもアリ**（フォロワー獲得＋誘導、2投稿する）

## Q7. note作成の苦手意識・AI感が強い

**相談内容:**
- AIと作っていいと思っても翌日「誰が読むんだろう」となる
- 販売したが売れたことがない
- AI感が強いと自覚しているが直すポイントが分からない

**回答まとめ:**
- **分析する目がまだ育っていない**のが根本原因
- 今はAIと向き合うフェーズではなく、**市場を見るフェーズ**
- **やるべきこと:** 売れているnoteの無料部分を10個以上分析する
  - この行は理想の未来、この行は価格訴求、ここで共感、ここでセールス…と分解
  - なぜ売れているのか考える
- 売れるには**3つのマッチ**が必要:
  1. 市場に流れているもの
  2. 顧客の声
  3. 自分のこと

### 💡 アイデア・気づき
- リサーチ・分析の癖をつけることが売れるnoteへの近道
- 自分と向き合うことも大事だが、外の市場理解とバランスよく

## Q8. みみちゃんの1日のタイムスケジュール

**回答まとめ:**
- 9時起床 → 朝食・家事 → 子供と遊ぶ → 昼食 → 昼寝 → 支援センター（14〜17時）or お出かけ → 夕食準備・風呂 → 21〜22時寝かしつけ → 22時旦那帰宅 → 夕食・晩酌
- **仕事は旦那が寝た後:** 早くて23時、遅いと0〜1時から開始
- 朝まで作業する日もある（4〜5時）が9時には起きる（最低4時間睡眠）
- **平日は旦那が昼までいる日** → その間にLINE返信など
- 昼間はパフォーマンス落ちる（夜型）
- ローンチ時は1週間毎日朝まで → その後1週間は控えめにバランス調整
- **1日あたり:** 1時間〜数時間（日によりけり）

## Q10. 「リスト」とは？ファン化と同じ？

**回答まとめ:**
- リスト ＝ **顧客リスト**（ファン化とは別物）
  - 見込み顧客・購入者の連絡先（名前・メアド・LINEなど）
- **一番取りやすいのはLINE**
  - 登録してもらえればブロックされない限りメッセージ配信可能
  - Threadsを見ていなくてもLINEは見てくれる可能性
- **購入者リスト**も重要
  - 例：note購入特典をLINEで渡す → 購入者のLINEリストが溜まる → アップセル（上位プラン）のセールスに使える
- リストがあると商品が売れやすくなる
- SNSだとLINEをリストと呼ぶことが多い。上級者はメルマガも活用
- **リアルの例:** イオンの赤ちゃん写真イベント → アンケート＋LINE追加 → 後日セールス送付

### 💡 アイデア・気づき
- みみ自身も「リストについてnoteにまとめたい」と言っている（よく質問される内容）

## Q11. 占いジャンル：無料→有料鑑定の案内文が刺さらない

**相談内容:**
- 無料鑑定後に「有料希望」と言われるが、その先に進まない
- コース2つ、案内文を相談者ごとにカスタマイズ
- 文が固い、無料鑑定と内容が被る、コンセプトに魅力がないのか…

**回答まとめ:**
- コンセプトに魅力がなければそもそも「有料希望」と言われないはず → **コンセプトは問題ない**
- **問題は案内文**
- 構成自体は良い（寄り添い → 恐怖 → 安心 → 案内）
- **改善ポイント:**
  1. 「この施術で分かること」の部分をもっと**刺さるように**（カスタマイズ強化）
  2. 「何が知れるか」ではなく**「どうなるか」**を伝える（相談者は答え＝理想の未来が知りたい）
  3. もう一段深く踏み込む
  4. 全体的に**文章が難しい** → もっと分かりやすく・読みやすく
- リサーチはしっかりしている印象。あとは文章の難易度調整と訴求の深さ
