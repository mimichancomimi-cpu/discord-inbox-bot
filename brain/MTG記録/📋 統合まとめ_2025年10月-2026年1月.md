### 1. mimiメソッド — 商品開発の軌跡

#### コンセプト・設計（10月）
- 商品名「mimiメソッド」に確定（10/7）
- 動画教材（1本5〜10分）× マインドマップ解説スタイル
- 構成: 基礎編 → リサーチ編 → 投稿作成編 → 運用編 → マネタイズ編
- 差別化: 動画教材（vs おきるママのテキスト）、ママ向け親しみやすさ（vs シロウさん）、リサーチ実演動画
- ワークシートはGoogleドキュメント形式（スマホのママ層に配慮）
- 「耳だけで聴ける」がママ層への最大の訴求ポイント

#### GPTs — 目玉機能
- メインのGPTs 1つに統合（章ごとに分けない）
- 2つのモード: 何でも相談 / ワーク形式
- ワーク形式に「強制力」を持たせる（脱線を元に戻す）
- ワーク完了後にアカウント設計シートを自動出力（スプシにコピペ可能）
- 「みみちゃんが伴走してくれる」体験としての訴求が強い

#### 価格戦略
- 一般価格: 19,800円
- つきのわハウスメンバー先行価格: 5,000円（アフィリエイト権利付き）
- 分割払いの訴求が効果的（「5,000円のコンテンツ4つ買うよりこっちの方が中身濃い」）
- 先行購入者は1件成約で約10,000円のリターン → 1件でペイ

#### ローンチの流れ
- 11/28〜30 の3日間限定セール
- LP公開: 11/28 19:00
- 決済: Utage（カード + 銀行振込）
- アフィリエイター: 先行購入メンバー + みみ直接依頼の限定
- LINE集客: 販売直前で約300人（前回超え）
- ゲストスピーカー（しろひげちゃん）の体験談がQ&Aの説得力を向上

#### ローンチ後（12月〜）
- 追加コンテンツ計画: ノートの書き方 / AI活用法 / 市場ギャップの見つけ方 / Threads基本操作ガイド

### 4. AI活用

#### みみ流・AI投稿作成
1. 書きたい内容は全部自分で考える → AIにはまとめ・整理・整えを任せる
2. 最初に書きたいこと全部を一気に伝える（タイトル・テーマ・言いたいこと・着地点）
3. AIが勝手に補足した部分は指摘して修正指示
4. 繰り返しでAIが自分に馴染んでいく

#### アカウント初期のAI育て方
1. まず自分で投稿を作る
2. 反応が良かった投稿だけをAIに学習させる
3. AIに投稿を作ってもらう → リライト
4. リライト後もAIに学習させる
5. 繰り返すとそのまま使えるレベルに

#### AI活用の3パターン
| パターン | 内容 |
|----------|------|
| 効率化 | 時短・作業スピードアップ |
| アイデアを広げる | 発想の幅を広げてもらう |
| 思考の整理 | 考えていることをまとめ・分かりやすい文章に |

#### AI利用の注意点
- 信頼度は50〜60%。基本的に間違えるものという前提で使う
- 苦手: 計算・数字、薬機法・著作権、情報の鮮度、ハルシネーション、専門用語、ツール操作手順
- AIとやり取りするだけで「考えた気」「やった気」になる危険
- 1アカウント = 1チャットで運用（内容の相違が出ない）
- 音声入力が非常に有効（AI活用の性能が上がる）

#### AIツール比較
| ツール | 特徴 |
|--------|------|
| ChatGPT | メイン使用。スマホで一番使いやすい |
| Gemini | 投稿の質が好み。Google連携で便利 |
| Claude | 賢い・ホスピタリティあり（未使用→試す予定） |
| Perplexity | 最新情報に強い |
| Cursor + Obsidian | 複数アカウント運用者に有効。データ管理 + AI指示が一体化 |

#### AI操作の危険性（実例）
- あんこがAIの指示通りにやったらLINE公式アカウントを削除してしまった（12/17）
- AIの回答を鵜呑みにせず、特に削除系操作は必ず人間が確認

### 6. Threads運用ノウハウ

#### リサーチのコツ
- 時間を決める（10分だけ等）
- 目的に合った投稿を3つ見つける → ミックスして新投稿を作る
- 「今日はフックを集める」等テーマを絞ると効率的

#### 投稿のルール
- 1日最低1投稿、多くても3投稿
- 予約投稿は使わない（伸びが悪くなる可能性）
- 下書きにストック → 投稿時にリライト
- ジャンル認知のためにキーワードを必ず入れる

#### フォロワーを増やすコツ
- **予告（ティザー）投稿が最強:** 「フォロワーさん限定で配布するのでフォローしてお待ちください」
- バズってもフォロワーが増えない場合 → プロフィールか固定ポストに問題
- 投稿で全部言いすぎない → **余白を残す**（もっと知りたいと思わせる）
- 距離感は近すぎない方がいい

#### AI投稿について
- 3〜5行の短い投稿ならAIそのままでもOK
- 長い投稿だとAIっぽさが出やすい
- AI感のある文章もバズるようになってきている
- ジャンルに合った単語が入っているかチェック

#### アカウントBAN対策
- シャドーバン: 3日間放置で復活することが多い
- BAN後: 1週間は見る専門（安全ユーザーアピール）
- 同じ名前・アイコン・投稿での再作成はNG
- オールドアカウント購入は非推奨（5個中4個がログイン時点でBAN）
- アカウント切り替えは最大10個まで問題なし

## 💡 全MTGから抽出したアイデア集

### 商品・コンテンツ
- マインドマップは全部完成させてから一気に撮影 → 並行してページ作成可能
- 動画のクオリティより中身が重要。テンポよくカット編集で十分
- リサーチ実演動画は他教材にない差別化ポイント
- GPTsでアウトプット → ドキュメントにメモの流れが実践的
- 「稼ぐ系なのか引かせぐ系なのか」を教材内で明確に分ける
- 超初心者はThreadsのアプリ操作自体に不安がある → 基本操作の動画コンテンツ追加が効果的
- ノウハウコレクター層に「基礎を一つの教材で完結できる」訴求が効果的

### マーケティング
- 事前育成済みアカウント付き講座は大きな差別化要素
- GPTs × 外注の組み合わせで大量アカウント運用が現実的に
- セールスレターは長ければ長いほど良い（飛ばし読みでも伝わるように構成）
- ゲストスピーカー（実践者の声）がQ&Aの説得力を大幅に向上させる
- 「何を売ればいいか分からない」層が意外と多い → 商品設計の基礎コンテンツの需要あり

### コミュニティ
- コミュニティ内の多様な専門性が最大の差別化ポイント
- 新メンバーケア: 雑談部屋でメンション → 疎外感を減らす
- 日報は1行でもOK。Threadsに投稿するぐらいの気軽さで

### AI・技術
- AIが当たり前の時代だからこそ「自分らしさ」が最大の価値
- AIに調べさせるより、自分で調べた情報をAIに読ませる方が正確
- 音声入力の活用でAIへの指示の質が大幅に向上
- AI生成文章が社会に浸透 → AIが書いた情報をAIが参照する循環問題が進行中
- Utageへの直接埋め込みは手間だが、検索性・UXの面で外部リンクより優れる

### 運用
- ノートはPayPay/Amazon Pay対応で主婦層に買いやすい → ノート優先
- バズらせたアカウントの裏側をチームで回す → 外注ノウハウとしてもコンテンツ化可能
- ノート1年記念の振り返り投稿は低単価ノートにできる
- Canvaのスライドで枠がズレる → ロック機能で改善
