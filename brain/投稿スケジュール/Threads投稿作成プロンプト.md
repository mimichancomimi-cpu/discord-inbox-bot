# Threads投稿作成プロンプトテンプレート

このプロンプトテンプレートを使用して、AI（ChatGPT、Claude等）にThreads投稿を生成させることができます。

```
## Threads投稿作成の要件

### 基本構造（3つの要素）

1. **フック（最初の一文）**
   - 目を引くワードを入れて、「え？」と思わせる
   - 共感を呼ぶ一言または問いかけ
   - 具体的な数字、状況、会話を入れる

2. **本題（ストーリー）**
   - 具体的なエピソードや状況を入れて、共感を引き出す
   - 体験談、あるある、失敗談など
   - 改行を活用して読みやすく

3. **問いかけ＆行動喚起で締める**
   - 読者がコメントしたくなる余白を作る
   - 「これって私だけ？」「みんなはどう？」などの問いかけ
   - CTA（行動喚起）を含める

### 重要なポイント

- **コメント数を増やすことが最優先**（いいね数より重要）
- **潜在率を高める**（長く読まれる構成にする）
- **会話が続く投稿**を作る（Threadsのアルゴリズムに最適化）
- **口語調**で親しみやすく
- **改行を効果的に使用**（4〜5行以上になる場合は改行しない）
- **文字形態のバランス**（漢字・ひらがな・カタカナ・絵文字・記号）
- **具体的なエピソード**（数字、場所、会話）を含める
- **ターゲットに届く単語**を入れる

### 投稿の長さ

- 短文: 1〜3行（断言型、問いかけ型）
- 中文: 3〜6行（エピソード型、論理展開型）
- 長文: 6行以上（詳細な体験談、ストーリー型）

### 絵文字の使用

{絵文字の使用方針}

### 避けるべき表現

{避けるべき表現}

## 過去の投稿例（参考）

{過去の投稿例（3〜5個）}

## 作成する投稿の要件

- テーマ: {テーマ（任意）}
- 投稿タイプ: {短文/中文/長文}（バランスよく）
- 絵文字: {使用する/使用しない}
- 文字数: {200-400文字程度（目安）}

## 出力形式

実際にThreadsに投稿する内容のみを出力してください。
- 分析や説明は不要
- マークダウン形式は使わず、プレーンテキストで
- 改行は実際の投稿と同じように

重要：
1. **過去の投稿の使い回しは禁止**。過去投稿をコピー・転用・言い換えだけした再利用はせず、**毎回あらためて**新しい内容を生成してください
2. **アカウントの内容をもとに柔軟に作成**。アカウント設計・コンテンツの柱・トーンを踏まえ、新しいテーマ・切り口・エピソード・表現で投稿を作成してください（スタイルや型は参考可）
3. アカウント設計の分析部分や説明は含めず、実際の投稿内容のみを生成してください
```

## プロンプト生成スクリプト

アカウント設計ファイルから自動的にプロンプトを生成するスクリプトも作成できます。

### 使用方法

1. アカウント設計ファイル（.md）を指定
2. テーマや投稿タイプを指定（オプション）
3. プロンプトを生成してコピー
4. ChatGPTやClaudeに貼り付けて投稿を生成

---

## プロンプトのカスタマイズポイント

### 1. テーマを指定する場合

```
- テーマ: {具体的なテーマ}
```

例：
- テーマ: ポイ活で月5万貯める方法
- テーマ: アラフォー母ちゃんの日常あるある
- テーマ: 執着を手放す方法

### 2. 投稿タイプを指定する場合

```
- 投稿タイプ: {短文/中文/長文}
```

### 3. 複数の投稿を一度に生成する場合

```
以下の要件で投稿を5個生成してください：
- 短文: 2個
- 中文: 2個
- 長文: 1個
```

### 4. バズを狙う場合

```
特に以下のポイントを重視してください：
- コメント数を増やすための問いかけを必ず含める
- 「これって私だけ？」「みんなはどう？」などの共感を呼ぶ問いかけ
- 具体的な数字やエピソードを含める
```

---

## 注意事項

1. **過去の投稿の使い回しはNG**: 過去投稿のコピー・転用・言い換えは禁止。毎回アカウントの内容をもとに**柔軟に新規作成**する
2. **アカウント設計に忠実に**: トーン&マナー・コンテンツの柱を守りつつ、新しい切り口・エピソード・表現で書く
3. **Threadsのアルゴリズムを意識**: コメント数を増やす工夫をする
4. **ターゲットに届く言葉を使う**: キーワードを自然に含める
