## 📈 エンゲージメント率の計算

```
エンゲージメント率 = (いいね数 + 返信数 × 2) / 表示数
```

- 返信数は2倍の重み付け（コメントが重要）
- 目標: 3%以上

## 🔍 成功パターンの分析方法

1. **パフォーマンス分析スクリプトを使用**
   ```bash
   python3 投稿スケジュール/パフォーマンス分析.py
   ```

2. **成功投稿の特徴を抽出**
   - エンゲージメント率が高い投稿を分析
   - 共通パターンを発見

3. **プロンプトに反映**
   - 成功パターンをプロンプトに自動追加
   - 新しい投稿生成時に活用

## 💡 改善のサイクル

1. **投稿を生成・公開**
2. **パフォーマンスを記録**
3. **パフォーマンス分析を実行**
4. **成功パターンを抽出**
5. **プロンプトに反映**
6. **次の投稿に活用**

このサイクルを繰り返すことで、継続的に改善できます。

## 📝 チェックリスト

投稿作成時に確認すべき項目：

- [ ] 問いかけが含まれているか
- [ ] CTA（行動喚起）が含まれているか
- [ ] 適切な文字数（200-400文字）か
- [ ] 具体的なエピソードが含まれているか
- [ ] アカウント設計のトーン&マナーに合っているか
- [ ] キーワードが自然に含まれているか
- [ ] 絵文字の使用方針に合っているか
