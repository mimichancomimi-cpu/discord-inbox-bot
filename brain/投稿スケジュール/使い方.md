# 投稿スケジュールスクリプト

投稿をスプレッドシートに転記して日時設定まで行う汎用スクリプトです。どのアカウントでも使用できます。

## 機能

- 投稿をスプレッドシートに転記
- 日時設定（1日2回：朝と夜）を同時に行う
- config.jsonのaccount_mappingsから自動的にスプレッドシートIDを取得
- どのアカウントでも使用可能

## 使用方法

### 方法1: Pythonスクリプトから直接呼び出す

```python
from 投稿スケジュール import schedule_posts_to_spreadsheet

# 投稿内容のリスト
posts = [
    "投稿1の内容",
    "投稿2の内容",
    # ...
]

# スプレッドシートに転記して日時設定
schedule_posts_to_spreadsheet(
    account_file_path="５０アカウント運用/12.天音🪽元恋愛依存が語る幸せの恋愛術.md",
    posts=posts,
    morning_hour=9,      # 朝の投稿時間（時）
    morning_minute=0,    # 朝の投稿時間（分）
    evening_hour=21,     # 夜の投稿時間（時）
    evening_minute=0,    # 夜の投稿時間（分）
    start_date=None      # 開始日（Noneの場合は今日）
)
```

### 方法2: コマンドラインから実行

#### 投稿を直接指定する場合

```bash
python3 投稿スケジュール/投稿スケジュール.py "５０アカウント運用/12.天音🪽元恋愛依存が語る幸せの恋愛術.md" \
  --posts "投稿1の内容" "投稿2の内容" "投稿3の内容"
```

#### 投稿ファイルから読み込む場合

```bash
# テキストファイル（1行1投稿）
python3 投稿スケジュール/投稿スケジュール.py "５０アカウント運用/12.天音🪽元恋愛依存が語る幸せの恋愛術.md" \
  --posts-file posts.txt

# JSONファイル（配列形式）
python3 投稿スケジュール/投稿スケジュール.py "５０アカウント運用/12.天音🪽元恋愛依存が語る幸せの恋愛術.md" \
  --posts-file posts.json
```

#### オプション指定

```bash
python3 投稿スケジュール/投稿スケジュール.py "アカウントファイル.md" \
  --posts-file posts.txt \
  --morning-hour 9 \
  --morning-minute 0 \
  --evening-hour 21 \
  --evening-minute 0 \
  --start-date 2026-01-20
```

## パラメータ

### schedule_posts_to_spreadsheet関数

- `account_file_path` (必須): アカウント設計ファイルのパス（親ディレクトリからの相対パス）
- `posts` (必須): 投稿内容のリスト
- `morning_hour` (オプション): 朝の投稿時間（時、デフォルト: 9）
- `morning_minute` (オプション): 朝の投稿時間（分、デフォルト: 0）
- `evening_hour` (オプション): 夜の投稿時間（時、デフォルト: 21）
- `evening_minute` (オプション): 夜の投稿時間（分、デフォルト: 0）
- `start_date` (オプション): 開始日（datetime.dateオブジェクト、Noneの場合は今日）

## 動作の流れ

1. config.jsonからアカウント名に対応するスプレッドシートIDを取得
2. Google Sheetsに接続
3. シートのヘッダー行を検索して列マッピングを取得
4. 投稿を1日2回（朝と夜）で予約
5. 投稿内容と日時を同時にスプレッドシートに書き込む

## 注意事項

- `config.json`にアカウント名とスプレッドシートIDのマッピングが必要です
- `credentials.json`（Google認証情報）が必要です
- スプレッドシートのシート名は「自動投稿」がデフォルトです（config.jsonで変更可能）
- アカウントファイルのパスは親ディレクトリ（Obsidian Vault）からの相対パスで指定します

## トラブルシューティング

### アカウントが見つからない場合

- config.jsonの`account_mappings`にアカウント名が登録されているか確認
- アカウント名の部分一致でも検索されます（例: "天音"で"天音🪽元恋愛依存が語る幸せの恋愛術"を検出）

### 列が見つからない場合

- スプレッドシートのヘッダー行に「ポスト文」「予約投稿日時」「予約投稿時間(時)」「予約投稿時間(分)」が含まれているか確認
- ヘッダー行は自動検出されますが、4行目以降にある場合は検出されない可能性があります

### パスが見つからない場合

- アカウントファイルのパスは親ディレクトリ（Obsidian Vault）からの相対パスで指定してください
- 例: `"５０アカウント運用/12.天音🪽元恋愛依存が語る幸せの恋愛術.md"`
