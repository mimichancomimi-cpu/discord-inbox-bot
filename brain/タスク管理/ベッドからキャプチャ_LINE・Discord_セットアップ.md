# ベッドからキャプチャ（LINE / Discord）セットアップ

PCは別室・CursorはPC専用、という前提で、**寝ながら思いついたことを逃さず取る**ための仕組み。

## いまの運用方針（これでOK）

| ツール | 役割 |
|--------|------|
| **Discord** | **X（Twitter）みたいに、ひたすらアイデア・考えを投げる。** 体裁は気にしない。あとでPCで整理する。 |
| **LINE** | **リマインダーが主**（「明日これやる」「締切」など）。通知で思い出す用。 |
| **ショートカット** | **使わなくてよい。** やりたくなったときだけ下の「任意」を見る。 |
| **本気の自動化** | **Discord → GitHub の md に自動追記**（Bot）。手順は **[[Discord自動化_GitHub連携_手順]]**（Git と常時起動が必要）。 |

**iCloud の Obsidian** は LINE／Discord から直接は触れないので、手運用なら **「スマホで投げる → PCで整理」**。自動なら **GitHub を受け皿**にする。

---

## 全体の流れ（メインルート）

1. **ベッド**：Discord の自分用チャンネルに **思ったまま送る**（短文・長文どちらでもOK）  
2. **LINE**：用事のリマインドだけ送る・予定に残す（運用は自分の好きな形で）  
3. **PCに着いたら（週に数回でもOK）**：Discord を上にスクロールしながら、必要なものだけ  
   - タスクっぽいもの → **`[[Inbox]]`**（運用ルールの1行形式に直す）  
   - ナレッジ・投稿ネタ → フォルダ `brain/00_メモ・下書き/` やアカウント別フォルダへ  
   - Cursor用 → そのままプロンプトに使う／メモに1行残す  

Cursor でナレッジを触るのは、PCに着いてからで大丈夫。

---

## A. Discord のセットアップ（メイン：呟き用）

### 考え方

- **整えなくていい。** 検索も効くので、あとから「あのときなんて書いたっけ」で辿れる。  
- チャンネルは **1本だけ** でもよい（`#つぶやき` など）。分けたければ `#アイデア` / `#愚痴` など好きに。

### チェックリスト（初回だけ）

- [ ] **サーバーを作成**（非公開・自分だけでOK）名前例：`さいきんのメモ`
- [ ] **テキストチャンネルを1つ**（または好きなだけ）  
  - 例：`#ベッド` または `#つぶやき` … **ここに無限に投げる**
- [ ] （任意）**ピン留め**：「ルールなしでOK・PCで整理」と1行書いておくと、未来の自分が安心する
- [ ] 通知：**@メンションのみ** など、夜鳴らない設定に
- [ ] アプリを **ホーム画面から開きやすい場所** に

### ピン留め用テキスト（任意・短くでOK）

```
この部屋はメモ用。体裁不要。PCでObsidianに整理する。
（ラベル付けしたい人だけ：T=タスクっぽい / M=メモ / C=Cursorでやりたい）
```

### Discord でできること・できないこと

| できる | できない（公式だけ） |
|--------|----------------------|
| Xみたいに連投・長文・画像 | iCloud の Vault を直接編集 |
| あとから検索 | |

## B. LINE のセットアップ（リマインダー主）

### 考え方

- **「いつかやる」より「いつ思い出すか」** をLINEに任せるイメージ。  
- トークの **自分宛** に「明日Inbox整理」と送る、**LINEの予定・リマインダー**、**Keepにチェックリスト** など、好きな道具でよい。

### チェックリスト（例）

- [ ] **リマインダー用**：繰り返し「金曜PCでDiscord→Inbox」など1本入れる（LINE・アラーム・カレンダーどれでも可）
- [ ] （任意）自分トークに **締切ワード** だけ投げる（Discordと役割を分けたいとき）

### LINE でできること・できないこと

| 向いていること | 向いていないこと |
|----------------|------------------|
| 通知で思い出す・期限 | アイデアの長期ストック（検索は **Discord の方が向く** ことが多い） |

---

## C. PCでの整理（Discord → Obsidian）

1. Discord を開き、**未整理の範囲**を見る（前回どこまでやったか、心の目印を1つ決めると楽）  
2. ピックアップしたものを  
   - **タスク** → `[[Inbox]]`（`README_運用ルール` の1行形式）  
   - **ネタ・引用・構成メモ** → `brain/00_メモ・下書き/` やアカウント別フォルダ  
3. **もう使わない呟き**は Discord に残したままでもよい（ストックとして）／消してもよい  

**転記用コピペ例（Inbox）：**

```markdown
## 2026-03-25（Discordから転記）

- [ ] （タスク名）｜種別:仕事｜領域:xxx｜優先度:P2｜状態:inbox｜期限:YYYY-MM-DD
```

**楽をするなら**：Discordの該当メッセージをまとめてコピーし、Cursorに「これをInbox形式とメモ振り分けの案にして」と頼む。

## D. ショートカット（任意・使わなくてOK）

**使わない運用で問題なし。** iPhoneから直接 `.md` に追記したい人向け。

やることの概要：コピー → ショートカット1回 → `[[ベッド自動追記]]` の末尾に追記。

### 初回だけ（ショートカットアプリ）

1. **ショートカット** → 「＋」で新規  
2. アクションを順に追加  

   1. **クリップボードの内容を取得**  
   2. **日付のフォーマット**（入力：現在の日付、形式：`yyyy-MM-dd HH:mm` など）  
   3. **テキスト** … 例：`- [ ] ［ベッド］` + クリップボード + ` ｜記録:` + 日付  
   4. **ファイルに追記** … `brain/タスク管理/ベッド自動追記.md` を指定（iCloud上のボールトから選択）、UTF-8  

3. 名前を付ける → ホーム画面に追加（任意）

### 注意

クリップボード空だと変な行が付く。**PCと同時に同じファイルを開いていると競合**しうる。

## E. Discord → GitHub 自動追記（やるならここ）

**Discord に送るだけで `[[Discord自動受信箱]]`（GitHub上）に1行追記**する。コードと手順はすでに Vault にあります。

- **手順書（チェックリスト付き）**：**[[Discord自動化_GitHub連携_手順]]**  
- **プログラム**：`brain/scripts/discord_inbox_bot/`（`README.md` も参照）

前提：**プライベート GitHub リポジトリ** と **Bot を動かしっぱなしにできるマシン or ホスティング**。iCloud だけ運用のままでは使えないので、**Git を正にするか** 受信箱だけ別リポジトリにするかを決める必要があります。
