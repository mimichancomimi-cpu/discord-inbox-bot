# タスク管理ルール（全アカウント＋プライベート共通）

このフォルダは、仕事とプライベートのタスクを一元管理するための場所。

## 使い方（最短）

1. 新しい依頼・思いつきは、まず `Inbox.md` に追記する  
2. 1日1回、`Inbox.md` から `今週フォーカス.md` に移す  
3. 実行するときは `アカウント別_タスク一覧.md` か `プライベート_タスク一覧.md` に詳細を残す  
4. 完了したら `done` にして、必要なら実績メモを1行残す

## ステータス定義

- `inbox`: 受け取っただけ（未整理）
- `next`: 次に着手する候補
- `doing`: 実行中
- `waiting`: 誰かの返答待ち
- `done`: 完了
- `someday`: 今はやらないが保留

## 優先度定義

- `P1`: 今週の成果に直結（最優先）
- `P2`: 今週中に進めたい
- `P3`: 時間があれば進める

## タスク記入フォーマット

以下の形式で統一する。

`- [ ] タスク名｜種別:仕事/私用｜領域:mimi/シナちゃん/つきのわハウス/家/その他｜優先度:P1/P2/P3｜状態:inbox/next/doing/waiting/someday｜期限:YYYY-MM-DD`

完了時はチェックを付け、状態を `done` にする。

## 運用のコツ

- タスクは短く具体的に書く（30分〜2時間で終わる粒度）
- 大きい作業は3つに分解して登録する
- 期限が未定でも一度入れる（あとで調整）
- 迷ったら `Inbox.md` に入れてから整える

## ベッドからのキャプチャ（LINE / Discord）

PCが別室で、寝る前にスマホだけ触れるとき用。

- **Discord**：**Xみたいに呟きで投げる**（アイデア・考え）。あとでPCで `Inbox` やメモに整理。  
- **自動化**：Discord → **GitHub の `[[Discord自動受信箱]]` に自動追記**するなら **[[Discord自動化_GitHub連携_手順]]**（Git＋Bot）。  
- **LINE**：**リマインダー主**（期限・思い出し）。  
- **ショートカット**：**なくてOK**。使うなら **`[[ベッド自動追記]]`** へ追記（任意）。
