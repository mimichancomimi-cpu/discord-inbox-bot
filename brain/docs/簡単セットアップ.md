# 🎯 簡単セットアップ（最初の一歩）

## 📌 まず最初にやること

### ステップ1: Google Cloud Consoleを開く

1. ブラウザでこのリンクを開いてください：
   👉 https://console.cloud.google.com/

2. Googleアカウントでログインします

---

### ステップ2: プロジェクトを作成する

1. 画面の上の方にある「プロジェクトを選択」というボタンをクリック
2. 「新しいプロジェクト」をクリック
3. プロジェクト名に「threads-poster」と入力（何でもOK）
4. 「作成」ボタンをクリック
5. 少し待つ（10秒くらい）

**これで完了！** ✅

---

### ステップ3: APIを有効にする（2つ）

#### 3-1. Google Sheets API

1. 左側のメニューから「APIとサービス」→「ライブラリ」をクリック
2. 上の検索ボックスに「Google Sheets」と入力
3. 「Google Sheets API」をクリック
4. 青い「有効にする」ボタンをクリック
5. 少し待つ

#### 3-2. Google Drive API

1. 同じページで「Google Drive」と検索
2. 「Google Drive API」をクリック
3. 「有効にする」ボタンをクリック
4. 少し待つ

**これで完了！** ✅

---

### ステップ4: サービスアカウントを作る

1. 左側のメニューから「APIとサービス」→「認証情報」をクリック
2. 上の「+ 認証情報を作成」ボタンをクリック
3. 「サービスアカウント」をクリック

4. 次の画面で：
   - サービスアカウント名: 「threads-poster」と入力（何でもOK）
   - 「作成して続行」をクリック
   - 「続行」をクリック（ロールはスキップでOK）
   - 「完了」をクリック

**これで完了！** ✅

---

### ステップ5: キー（認証情報）をダウンロード

1. 作成したサービスアカウント（一覧に表示されている）をクリック
2. 上のタブから「キー」をクリック
3. 「キーを追加」→「新しいキーを作成」をクリック
4. 「JSON」にチェックが入っていることを確認
5. 「作成」をクリック

**⚠️ 重要**: JSONファイルが自動的にダウンロードされます！
このファイルは**絶対に失くさないでください**。

---

### ステップ6: ファイルを配置する

1. ダウンロードしたJSONファイルを見つける（通常は「ダウンロード」フォルダ）
2. ファイル名を `credentials.json` に変更
3. このファイルを `/Users/sasaki/Documents/Obsidian Vault/` に移動

**確認方法**:
- Finderで `/Users/sasaki/Documents/Obsidian Vault/` を開く
- `credentials.json` というファイルがあるか確認

**これで完了！** ✅

---

## 🎉 ここまでで完了！

## 📝 次のステップ（スプレッドシートの設定）

### ステップ7: サービスアカウントのメールアドレスを確認

1. `credentials.json` をテキストエディタで開く
2. `"client_email"` という部分を探す
3. その後の `"xxxxx@xxxxx.iam.gserviceaccount.com"` というメールアドレスをコピー

**例**: 
```json
"client_email": "threads-poster@my-project.iam.gserviceaccount.com"
```
この `threads-poster@my-project.iam.gserviceaccount.com` の部分をコピー

---

### ステップ8: スプレッドシートに権限を付与

**ぽんこつ母ちゃんのスプレッドシートで試してみましょう：**

1. このスプレッドシートを開く：
   https://docs.google.com/spreadsheets/d/1uj5a4GVcxMTzW8aoCIQL8Oj9Y8qvY3Wsn3bYGfEPymw/edit

2. 右上の「共有」ボタンをクリック

3. ステップ7でコピーしたメールアドレスを貼り付け

4. 権限を「編集者」に設定

5. 「送信」をクリック

**これで完了！** ✅
