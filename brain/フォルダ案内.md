# ボールト構成の案内

Obsidian で `brain` を開いている想定です。一覧の先頭に出るよう、整理用フォルダには `00_` プレフィックスを付けています。

| `docs/` | 運用ナレッジ、セットアップ手順、**分析レポート**、**アカウントナレッジ統合プロンプト** |

| `投稿スケジュール/` | CSV 抽出スクリプトなど |
| `タスク管理/` | タスク・今週フォーカス・[[ベッドからキャプチャ_LINE・Discord_セットアップ]]・[[Discord自動化_GitHub連携_手順]]（Discord→GitHub自動）・[[Discord自動受信箱]]・[[ベッド自動追記]]（ショートカット任意） |

## 自動化まわり（`brain` 直下に残しているもの）

`scripts/` 内のツールは、作業ディレクトリを `brain` にしたときに `config.json` / `credentials.json` を読む想定です。**これらはフォルダに入れず直下のまま**にしています。

- `config.json` / `config.example.json`
- `credentials.json`
- `requirements.txt`
- `ゆるスピのアオ子_insights_cache.json`（インサイトキャッシュ）

## ひとつ上の階層（`Documents` ルート）

| 名前 | 内容 |
|------|------|
| `attachments/` | 画像の集約先（ウィキリンクは `![[attachments/…]]`） |
| `brain/` | 上記のメインボールト |
| `tools/` | スタエフ文字起こし拡張、`vault-image-utils`（画像集約スクリプト）など |
| `.cursorrules` | Cursor 用ルール |

画像集約スクリプトは `tools/vault-image-utils/README.md` を参照（**ルートは `Documents`** として動作）。

## 2025-03-25 の整理メモ

- 経理・BASE 系ノート → `00_経理_BASE/`
- 短いメモ類 → `00_メモ・下書き/`

Obsidian の `[[リンク]]` は通常ノート名で解決するため、上記移動でリンクが切れにくい構成にしています。パス指定のリンクがある場合は、検索でファイル名を探して差し替えてください。
