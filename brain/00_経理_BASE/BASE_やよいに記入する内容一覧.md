# BASE やよいに記入する内容一覧

**ルール**
- **売掛金・売上** ＝ **実際の売上**（手数料控除後の受取額）で計上する
- **手数料** ＝ **経費**（支払手数料）で計上する
- **毎月末** → その月の実際の売上を「売掛金 / 売上」で計上
- **振込日** → 着金を「普通預金 / 売掛金」で計上。振込手数料は「支払手数料（経費）/ 売掛金」で計上

## まとめ

| いつ | 何を記入するか |
|------|----------------|
| **毎月末** | **実際の売上**（手数料控除後）を「売掛金 ○○円 / 売上 ○○円」で1行 |
| **振込日** | 着金を「普通預金 ○○円 / 売掛金 ○○円」で記入。振込手数料は**経費**として「支払手数料 ○○円 / 売掛金 ○○円」で1行追加 |
