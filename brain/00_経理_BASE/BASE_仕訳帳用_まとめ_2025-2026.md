---

## 仕訳のルール

| 項目 | 内容 |
|------|------|
| **売掛金・売上** | **実際の売上**（手数料控除後の受取額）で計上する |
| **手数料** | **経費**（支払手数料）で計上する |

| タイミング | 仕訳 |
|------------|------|
| **毎月末** | 実際の売上 → 売掛金 / 売上 |
| **振込日** | 着金 → 普通預金 / 売掛金 |
| **振込日（手数料あり）** | 振込手数料は経費 → 支払手数料 / 売掛金 |
