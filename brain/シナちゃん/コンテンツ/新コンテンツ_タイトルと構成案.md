### 第1章：なぜ「AI×note」で、SNSなしでも売れるのか

**目的:** 不安を解消し、「この方法でいい」と腹落ちさせる。

| 節 | 内容イメージ |
|----|--------------|
| 1-1 | **SNS疲れの時代に、もう一つの選択肢** … 毎日投稿・いいね周りがなくても、noteは売れる。その前提の変化。 |
| 1-2 | **AIは「丸投げ」じゃない。役割分担の考え方** … あなた＝素材（体験・想い）、AI＝シェフ（構成・見せ方）。完全ガイドブックの「基礎」の上にAIを乗せる。 |
| 1-3 | **SNSなしで売れる理由** … note内で完結する設計（検索・タイトル・導線）。「置いておくだけ」で届く仕組みを一言で。 |

---

### 第2章：AIで「売れるnote」を書く｜素材の渡し方とプロンプト

**目的:** 何を自分で用意し、何をAIに任せるかを具体化する。

| 節 | 内容イメージ |
|----|--------------|
| 2-1 | **AIに渡す「素材」とは** … 体験談・失敗談・悩み・想い。等身大の主婦のリアルがそのまま価値になる。 |
| 2-2 | **「人間臭く」書かせるコツ（共感ハック）** … AIに冷たい文章を書かせない。泥臭い感情を混ぜて、読者が「私のことだ」と思う書き方。 |
| 2-3 | **実践：プロンプトとワークフロー** … スマホだけでもできる手順。素材メモ→AIに渡す→整える、の流れ。具体例・テンプレあり。 |
| 2-4 | **よくあるつまずきと対処法** … 同じような文章になる、長すぎる、共感が弱い、など。 |

---

### 第3章：SNSなしで「届く」設計｜noteの中で完結させる

**目的:** 投稿しなくても、必要な人に読まれ・買われる設計を理解する。

| 節 | 内容イメージ |
|----|--------------|
| 3-1 | **「検索で見つかる」noteの作り方** … タイトル・キーワード・見出し。note内SEOの基本。 |
| 3-2 | **読者を「買う」まで誘導する導線** … 無料部分で共感→有料で解決。読了後の行動（購入・フォロー）を設計する。 |
| 3-3 | **放置でも売れ続ける仕組み** … 1本置いておくだけでは足りない部分、複数記事・導線の積み重ね。実験結果・数字の話。 |
| 3-4 | **スマホだけ・PCなしでもできる？** … ママ友事例（PCなし）を交えて、必要な環境を整理。 |

## 構成のポイント

1. **完全ガイドブックと被らない** … 第1章は「マインド・テーマの見つけ方」ではなく「なぜAI×SNSなしで売れるか」に特化。執筆の「型」は完全ガイドブックを参照しつつ、ここでは「AIの使い方」と「SNSなしの設計」に絞る。
2. **証拠を最初と所々に** … はじめに・第1章・第3章で、ママ友5万・自分の実験結果を入れる。
3. **すぐ動ける** … 第4章で7日間ステップにし、特典でプロンプト・チェックリストを渡す。
4. **長さの目安** … 完全ガイドブックが3万字超なので、今回は「2万字前後」で実践的にまとめると、読み切れる＋価格の差別化もしやすい。
