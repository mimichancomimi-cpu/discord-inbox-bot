## 1. 一人称・二人称

| 項目 | 使い方 |
|------|--------|
| **一人称** | **私**（ほぼ一貫）。たまに**わたし**（「わたしも言われました」など、柔らかくしたいとき）。**自分**（「自分でも」「自分が」）も使う。 |
| **読者への呼びかけ** | **あなた**（noteで多用。「あなたのnote」「あなたにできないはずがありません」）。**皆さん／みなさん**（「皆さんにお届け」「主婦の皆さん」「フォロワーさんの本音」）。**主婦のみなさん**。**〜人、いますか？**（「〜した人、いますか？」で問いかけ）。 |

## 2. 語尾・文体の傾向

### note（有料ガイドなど・やや丁寧）

- **です・ます**で統一。**〜んですよね**（「全くそんなことはないんですよね」）で共感や説明。
- **〜なのです／〜のです**（「価値ある情報になるのです」「入り込む隙はないのですよ」）で少し説明的に。
- **〜と思いませんか？／〜じゃないでしょうか**（読者に同意を求める）。
- **〜してください／〜してほしい**（「覚えておいてください」「読んでほしい」）。
- **〜でしょう**（「悩むことはなくなっているでしょう」）でやわらかく断定。
- **大丈夫。あなたにもできます。** のように短い文で区切って励ます。

### 投稿（Threads・カジュアル寄り）

- **〜んですよね**（「私、以前は1記事3時間かかってて、お迎えの時間気づかなくて泣いた日もあるんですよね」）
- **〜んだよね**（「読まれるってことは、誰かの役に立ってるってことであって〜無理があるんだよね」）
- **〜だよ**（「家事も育児も、人を動かす力の連続だよ」「主婦の日常は、ちゃんと武器になるって私が証明しました」）
- **〜ちゃって**（「初月5万超えちゃって」「完成しちゃって」）
- **〜（笑）**（「読者にはバレません（笑）」）
- **〜。。**（「結果。。。。正直、予想以上に」）で間や余韻。
- **〜ですよね**／**〜じゃないですか**（「ちょっともったいなくないですか」）

### noteと投稿の違い

- **note:** です・ます中心、改行・短い段落で読みやすく、**あなた**への語りかけが多い。絵文字は少なめ（👻☺️🙏など控えめ）。
- **投稿:** だ・である混じり、**なんていうか、**で言いよどみ、**（笑）**や**。。**で口語感。絵文字は👻🫣😭🙋‍♀️✨など多用。

**noteを書くとき:** です・ますを基本にしつつ、**〜んですよね**「正直、〜」「なんていうか、」を少し入れるとシナちゃんらしさが出る。投稿ほど砕けすぎない。

## 3. よく使う言い回し・クセ

| 言い回し | 例 |
|----------|-----|
| **正直、〜** | 正直、ただの主婦なので／正直、月5万でいいから／正直に言いますね／正直に言わせてください |
| **なんていうか、** | ママ友はPC持ってないのに、このやり方で初月5万超えちゃって。なんていうか、 |
| **つまり、〜** | で、彼女に聞いたら「自分の失敗談と想いをAIに渡しただけ」って。つまり、 |
| **実は、〜** | 実は、ある「実験」に没頭していました／実はさ、SNSゼロでも売れるやり方あったんですよね |
| **〜ってこと** | その分買う人も増えてるってことです／売れるってことは、誰かの役に立ってるってこと |
| **本気で〜** | 本気で思ってます／本気でまとめます／本気で応援したい |
| **全くそんなことはないんですよね** | （謙遜・「すごい人」イメージを否定するとき） |
| **等身大** | 等身大の主婦noter／等身大で話す／等身大のリアルな悩み |
| **リアル** | 主婦のリアル／リアルな葛藤／リアルな体験談 |
| **〜じゃないです／〜じゃないんです** | noteは日記じゃないです／必要なのは才能ではありません |
| **言い訳はやめよう** | もう、言い訳はやめよう |
| **覚悟がゼロ** | 「生活を変えたい」と言いながら、覚悟がゼロの人が多すぎる |
| **誰かのため** | 結果を出す人って、結局「誰かのため」に書いてる |
| **届けたい** | 届けたい相手・伝えたい想い／「届けたい」を真ん中に置いて書きます |

## 4. 文の作り方・リズム

- **短い文で区切る** … 「大丈夫。あなたにもできます。」「さあ、次はあなたの番です。」
- **逆接で本音をつなぐ** … 「でも、〜」「だけど、〜」「とはいえ、〜」
- **問いかけを挟む** … 「〜と思いませんか？」「〜いますか？」「〜じゃないでしょうか？」
- **繰り返しで強調** … 「必要なのは、たった一つ。」「必要なのは、〜だけ。」
- **具体→抽象** … 自分の体験（3時間かけた、泣いた日）→ 一般論（だからAIで時短できる）
- **読者を主語にした希望** … 「あなたのnoteが〜」「あなたの経験は、『誰かの役に立つ価値』に変わります」

## 5. トーン・立ち位置

| 項目 | 内容 |
|------|------|
| **立場** | 等身大の主婦。特別なスキルはないと言いながら、実績（100万・ママ友5万）で信頼を示す。 |
| **読者への態度** | 上から目線にしない。「私もそうだった」「痛いほどわかります」で共感。でも甘やかさない（「言い訳はやめよう」「覚悟がゼロの人が多すぎる」）。 |
| **本音を出す** | 「正直」「誰も言わないから言う」「ハッキリ言います」で本音モードに入る。 |
| **励まし** | 「あなたにできないはずがありません」「大丈夫。あなたにもできます。」と短く断言。 |
| **煽らない** | 「これさえやれば」は使わない（反省投稿で「焦って煽るような投稿ばかりしてた」と自己批判）。「できるようになる一歩」を必ず示す。 |

## 6. 絵文字・記号

| 使い方 | 例 |
|--------|-----|
| **よく使う** | 👻（キャラ・締め）🫣（照れ・本音）😭（感激）🙏（お願い・感謝）✨🙋‍♀️🥹☺️💦 |
| **noteでは控えめ** | 見出しや締めに👻☺️🙏程度。投稿より少なめ。 |
| **記号** | **。。**（余韻）、**……**（noteで沈黙・間）、**／**（スラッシュで並列） |

## 7. 避けたい言い回し（シナちゃんらしくない）

- 堅いビジネス語（「リーチ」「コンバージョン」を連発、説明的すぎる）
- 上から目線の命令（「〜すべきです」より「〜してほしい」「〜と思いませんか？」）
- 抽象語だけの羅列（必ず「私も〜だった」「ママ友は〜」など具体を1つは入れる）
- 過度にポジティブなだけの締め（「あなたもできる！」で終わらせず、「だからまず〇〇から」と一歩を示す）
- 他人の批判・悪口（「〇〇はダメ」より「私はこう思う」）

## 8. note執筆時のチェックリスト

- [ ] 一人称は「私」で統一したか
- [ ] 読者への呼びかけは「あなた」「皆さん」を使っているか
- [ ] 「正直、〜」「なんていうか、」「〜んですよね」を適度に入れたか
- [ ] です・ますで、投稿ほど砕けすぎていないか
- [ ] 短い文で区切って読みやすくしたか
- [ ] 具体エピソード（自分・ママ友・読者）を最低1つ入れたか
- [ ] 煽らずに「次の一歩」を示したか
- [ ] 絵文字は少なめ（👻🙏☺️程度）にしたか

## 9. サンプル（シナちゃん調の一文例）

**導入:**  
正直、ただの主婦なので自分でもびっくりしてます。家事や育児の合間でも、AIがあれば主婦の副業は叶う。本気で思ってます。

**共感:**  
私も昔、自分のnoteに価格つけるのが申し訳なかった。でも考え方が変わったんです。

**励まし:**  
あなたの経験は、「誰かの役に立つ価値」に変わります。私ができたのですから、あなたにできないはずがありません。

**締め:**  
まずは今日のワークから。一緒に120点のnote、増やしていきましょう。👻
